# Obsidian — Restoran Boshqaruv Tizimi (Frontend)

Bu loyihaning **Frontend** qismi hisoblanadi. U Vue 3, Vite, Tailwind CSS va Pinia yordamida qurilgan.

Loyiha haqida batafsil ma'lumot, skrinshotlar va ishga tushirish yo'riqnomalari bilan loyihaning asosiy [README.md](../README.md) faylida tanishishingiz mumkin.

## Ishga tushirish buyruqlari

```bash
# Kutubxonalarni o'rnatish
npm install

# Ishlab chiqish serverini ishga tushirish (dev server)
npm run dev

# Production uchun yig'ish (build)
npm run build
```
